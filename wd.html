Text editers

A good text editor should be simple to use and available on a variety of platforms. The choice is arbitary,
however, the following criteria are often used to make a choice:

  • -Produces pure text files
  • -WSIWYG - What-You-See-Is-What-You-Get
  • -Simple to use, yet powerful editing features
  • -Syntax highlighting to show tags/keywords
  • -Any of the text editers mentioned above fit these criteria

HTML


All websites are developed using HTML, also known as HyperText Markup Language.
The language describes how the information is to be displayed in a web browser.


In order to make information available on the www, you will need:

  • -A web Server or Service (e.g. github.com)
  • -A web page written in HTML (e.g. mypage.html)
  • -A web browser at the client (e.g. Google Chrome)

However nowadays, you would need more than just HTML to create a dynamic webpage. Here's how:

  • -CSS (Cascading Style Sheets) to format the information into a uniform, consistent and pleasing nature
  • -Scripting (e.g. Javascript) to add dynamic content to the information.
